<h1>インストール</h1>
<br>
インストールは簡単です。aptでできます。<br>
<pre class="prettyprint"><code class="prettyprint">
sudo apt install pv
</code></pre>
だけでインストールできます。<br>
<h1>使い方</h1>
<br>
ddコマンドに細工をします。<br>
修正前：dd if=a.img of=/dev/sda<br>
修正後：pv a.img | dd of=/dev/sda<br>
このようにコマンドを修正すると進歩状況付きでddコマンドを実行できます。
