<h1>早速ツールをインストール</h1>
<br>
早速やってみましょう。<br>
<pre class="prettyprint"><code class="prettyprint">
sudo apt install python3-pip
sudo pip install youtube-dl
</code></pre>
でインストールできます。
<h1>使い方</h1>
<br>
早速やってみましょう。<br>
ダウンロードしたいディレクトリに移動したら、youtube-dlコマンドにURLを引数として指定して実行してください。<br>
